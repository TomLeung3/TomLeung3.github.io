Mysql for Mac 忘记root密码

前言

目前mysql 8已经支持安装时设置root密码，但是在相对于在使用mysql 5.7的人来说安装时的默认密码好纠结（这密码太安全，完成记不住的）很容易忘记，至少博主不会去记住密码类型的人，行修改密码吧，Act Now

环境信息：

系统：macOS Sierra 版本 10.12.6

安装包：mysql-5.7.17-macos10.12-x86_64.dmg 链接:https://pan.baidu.com/s/19esUzXCpxyI2AZFhzHSwqw 密码:ftdr

工具：Navicat Premium 12.0.23

Act Now

停止Mysql 服务

sudo /usr/local/mysql/support-files/mysql.server stop

输入Mac密码

Password:【你的Mac密码】

Shutting down MySQL

. SUCCESS!

同时也你可以在系统偏好设置是停止

进行mysql安装根目录

cd /usr/local/mysql/bin

注意：请确认mysql根目录文件是否是mysql，博主的文件是mysql-5.7.17-macos10.12-x86_64，所以出现下面的错误。

切换到root权限

sudo su

输入Mac密码

Password:【你的Mac密码】

sh-3.2#

随后输入

./mysqld_safe –skip-grant-tables &

注意：接着会有下面的提示，但是不要在意，我们能从系统偏好设置中看到Mysql中看到重启启动了

2018-06-01T05:53:16.6NZ mysqld_safe Logging to ‘/usr/local/mysql/data/TomdeMacBook-Pro.local.err’.

2018-06-01T05:53:16.6NZ mysqld_safe Logging to ‘/usr/local/mysql/data/TomdeMacBook-Pro.local.err’.

2018-06-01T05:53:16.6NZ mysqld_safe Starting mysqld daemon with databases from /usr/local/mysql/data

进入mysql安装文件夹，打开mysql的命令文件

接着输入下面的，重新刷新权限表

FLUSH PRIVILEGES;

设置密码

SET PASSWORD FOR ‘root‘@’localhost’ = PASSWORD(‘你的新密码’);

找个工具登录试试
